## Key Design Decisions

### 1. Coherence-Dependent Metabolic Cost
The critical innovation linking coherence to survival:

```python
coherence_multiplier = 2.0 - 1.5 * coherence
# coherence=1.0 -> multiplier=0.5 (efficient)
# coherence=0.0 -> multiplier=2.0 (wasteful)
total_cost = (base_cost + action_cost) * coherence_multiplier
```

### 2. Coherence Calculation
Based on autopoietic_entity.py:
- Predictability (30%): Low variance in state changes
- Stability (30%): Low variance in recent states
- Complexity (20%): Moderate state variance
- Circularity (20%): Temporal autocorrelation

### 3. RNN Architecture
Simple but effective:
```
state_t+1 = tanh(W_rec @ state_t + W_in @ sensor)
action = tanh(W_out @ state_t+1)
```
